Água

A água é um dos recursos naturais mais importante para a humanidade, posto que não podemos viver se mela. Da mesma forma, os vegetais e animais precisam dela para viver.

A maior parte de cada uma das células de nosso corpo possui água. Usamos a água para beber, para o preparo das refeições, para a higiene e doméstica, ou ainda para a pesca, transporte, produção de energia elétrica, irrigação etc.
