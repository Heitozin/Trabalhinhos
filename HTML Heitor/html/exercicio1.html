Água - Original

A água é um dos recurso naturais mais importantes para a humanidade, posto que não podemos viver sem ela. Da mesma forma, os vegetais e animais precisam dela para viver.

A maior parte de cada uma das células de nosso corpo possui água. Usamos a água para beber, para o preparo das refeições, para a higine pessoal e doméstica, ou ainda para a pesca, transporte, produção de energia elétia, irrigação etc.
