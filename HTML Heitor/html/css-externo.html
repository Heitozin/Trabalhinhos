Tecnologia Web

Algumas das principais tecnologia e linguagens web são o HTML, CSS, JavaScript e PHP.

São linguagens usadas para estruturar, estilizar e implementar comportamentos dinânicos aos WebSites.

Também é muito comum o uso de sistemas gerenciadores de bancos de dados, como MySQL, na implementação de sites.

Abaixo você encontra uma lista com algums programas utilizados para o desenvolvimento de sites:

  • Brackets
  • Notepad++
  • Sublime
  • Dreamweaver

Desenvolvido de acordo com os Padrões Web.
